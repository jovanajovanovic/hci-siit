This program is intended to follow the teaching of the computer center. Below we explain the functionality of the program.

Home page

On this page shows the distribution in the form of a table. The table is divided into classrooms, each classroom to 6 working days of the week. For easier monitoring and better visual presentation there are options to filter the schedule by day or by classroom. On this page there is a toolbar with options File, View and Add.

Clicking on the option add or view opens a drop-down menu with four options, or the option to add or review items, directions, classrooms and software. Depending on which option you choose will open us to a certain window. All these windows can be opened by using the shortcut ().

Working with terms

Right-click on a cell of you, you will open a drop down menu where you can choose to add appointment and will open the window for adding the term. It is necessary that the list of subjects selected one of the items and if you want you can write a brief description. Click OK term will be added to the schedule. Appointments can be dragged on schedule. Clicking on dates again will open a form to add where you can change the details of the time. On the left side you there is an extra label with all cases and the number of terms added to the case and the number of necessary terms.

View subject, classroom, course or software

If you choose one of these four options, the name will open a window on which tabulated all the facilities for the selected entity. It is possible to group the object according to one of the columns and that grouping is performed so that the name of the column switch in the upper panel. Left click on the table header will open us a drop-down menu from which we can select any of the available options, such as sorting, filtering, filtering by columns, search. Clicking on the column chooser option will show us in the lower right corner of the window, where we can odabermo columns you want to show us. Double-clicking a row in the table opens a window for changing the data for the selected object. With a left-click on the red show us the drop-down menu where we can choose the Delete option with which deletes the selected object or Edit.

File

Clicking on this option will open a drop-down menu from which you can select the option to Open(Ctrl+O), Save(Ctrl+S), or Clear all. By selecting Open opens a window where you can select from a list of schedule you want to open. Clicking the Save option also opens prozogde you can enter the name of the schedule and to save it under that name in the database. Finally, clicking Clear all you can delete all data from the database.

Add subject

This is a page to add subject.

Id - in this field you can entered a unique code of subject. This field is necessary.

Name - in this field you can entered name of subject. This field is necessary.

Course - you can choose the course in which the subject is being performed.

Description - in this field, you should briefly describe the subject.

Projector - you need to select whether a projector is needed for the subject.

Table - you need to select whether a table is needed for the subject.

Smart table - you need to select whether a smart table is needed for the subject.

Group size - in this field you can entered a number of students in one group.

Length of terms - in this field you can entered a number of school hours required for one exercise.

Number of terms - in this field you can entered a number of terms required for one subject in one week.

Operation system - you need to select whether a operation system is needed for the subject.

Software - you need to select whether a software is needed for the subject.
